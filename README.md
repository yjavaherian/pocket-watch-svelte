# Pocket Watch

a webapp you can use to keep track of how much time you put on your projects.

## Stack

- pnpm (package manager)
- svelte (front library)
- sveltekit (fullstack library)
- drizzle ORM (ORM)
- vite
- typescript

## Development

```bash
pnpm install
npm run dev
```

## Building

```bash
pnpm run build
pnpm run preview
```
